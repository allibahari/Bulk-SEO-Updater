MetaMate is a lightweight and standalone WordPress plugin that lets you bulk update SEO meta titles and descriptions via Excel (.xlsx). Supports Yoast SEO and Rank Math, with Persian (Jalali) date logs.

== Description ==

MetaMate allows you to easily update hundreds of posts’ SEO meta titles and descriptions in bulk using an Excel file. This plugin supports the two most popular SEO plugins, Yoast SEO and Rank Math, and includes a Persian interface with Jalali date support.

Features:
* Bulk update SEO Meta Titles and Descriptions
* Upload Excel files with URLs and new metadata
* Supports both Yoast SEO and Rank Math
* Persian interface + Jalali date logs
* No Composer required — fully standalone
* View full change logs with timestamps

Ideal for SEO specialists, agencies, and website owners who want a fast, reliable bulk SEO updater.

== Installation ==

1. Upload the `metamate` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the MetaMate admin page.
4. Upload your Excel (.xlsx) file with URLs and new SEO data.
5. Review the change logs after update.

---

== نصب ==

1. پوشه `metamate` را در مسیر `/wp-content/plugins/` بارگذاری کنید.
2. افزونه را از طریق منوی افزونه‌ها در وردپرس فعال کنید.
3. به صفحه مدیریت متامت بروید.
4. فایل اکسل خود را با آدرس‌ها و اطلاعات جدید SEO بارگذاری کنید.
5. پس از آپدیت، گزارش تغییرات را بررسی نمایید.

== Frequently Asked Questions ==

= Does it support Persian URLs? =  
Yes, MetaMate fully supports Persian URLs in your Excel files.

= Can I update only titles or only descriptions? =  
Yes, you can leave either the title or description column empty to update only one of them.

= Which SEO plugins are supported? =  
Currently, Yoast SEO and Rank Math.

= Does it require Composer or external dependencies? =  
No, MetaMate is fully standalone.
